## Description

Each card has an emotion, it's your job to match the corresponding emotion.
Each move counts. Your rating will decrease the more moves you make.
Check the timer and bet your personal best!

## Features

-   Background Parallax
-   Mouse Pointer
-   Deck Shuffle
-   Timer
-   Reset Button
